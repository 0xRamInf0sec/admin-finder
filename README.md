# admin-finder
This tool is used to find the admin login page of a website.

# How to run
 git clone https://github.com/Ramalingasamy012/admin-finder.git<br/>
 cd admin-finder<br/>
 pip3 install requirements.txt<br/>
 python3 adminfinder.py<br/>
